# RunePulse Engine
A genre-agnostic 2D Game Engine template for Odin + SDL3.

## Features
* **SAT Physics**: Supports any convex polygon collision.
* **Input Buffering**: Handles single-press and held keys.
* **Modular Design**: Separate files for Render, Physics, and Input.

## How to Run
1. Install [Odin](https://odin-lang.org/) and [SDL3](https://github.com/libsdl-org/SDL).
2. Run: `odin run .`
